KakaoBig OTF 폰트 파일 위치 안내
====================================

이 폴더에 아래 3개 파일을 넣어주세요:

  KakaoBigOTF-ExtraBold.otf  (font-weight: 800)
  KakaoBigOTF-Bold.otf       (font-weight: 700)
  KakaoBigOTF-Regular.otf    (font-weight: 400)

폰트 파일이 없으면 시스템 기본 폰트(Apple SD Gothic Neo / 나눔고딕 등)로
대체 표시됩니다. 명함 PNG 출력 품질을 위해 OTF 파일 배치를 권장합니다.

파일명이 다를 경우 index.html 상단 @font-face src URL을 수정하세요.
